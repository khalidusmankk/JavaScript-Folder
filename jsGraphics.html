Plotly.js is a charting library that comes with over 40 chart types:

Students Result Chart

Pie Charts

To display a pie instead of bars, change x and y to labels and values, and change the type to "pie" as below

          const data = [
        {
          labels: classes,
          values: students,
          type: "pie",
          marker: { color: "red" },
        },
      ];

Donut Charts

To display a donut instead of a pie, add a hole

        const data = [{
  labels: xArray,
  values: yArray,
  hole: .4,
  type: "pie"
}];
