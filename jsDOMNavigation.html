DOM Navigation

With the HTML DOM, you can navigate the node tree using node relationships.

Everything in an HTML document is a node

   The entire document is a document node
   Every HTML element is an element node
   The text inside HTML elements are text nodes
   Every HTML attribute is an attribute node (deprecated)
   All comments are comment nodes

With the HTML DOM, all nodes in the node tree can be accessed by JavaScript. New nodes can be created, and all nodes can be modified or deleted.

The nodes in the node tree have a hierarchical (darjawar) relationship to each other. The terms parent, child, and sibling are used to describe the relationships.

In a node tree, the top node is called the root (or root node)

Every node has exactly one parent, except the root (which has no parent)

A node can have a number of children

Siblings (brothers or sisters) are nodes with the same parent

Attributes

All HTML elements can have attributes
The href attribute of < a > specifies the URL of the page the link goes to
The src attribute of < img > specifies the path to the image to be displayed
The width and height attributes of < img > provide size information for images
The alt attribute of < img > provides an alternate text for an image
The style attribute is used to add styles to an element, such as color, font, size, and more
The lang attribute of the < html > tag declares the language of the Web page
The title attribute defines some extra information about an element
