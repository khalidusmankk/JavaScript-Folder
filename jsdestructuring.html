Destructing

Destructuring is a powerful feature in JavaScript that allows you to extract values from arrays or properties from objects and assign them to variables in a concise and readable way.

Destructuring Assignment Syntax

The destructuring assignment syntax unpack object properties into variables:

let {firstName, lastName} = person

It can also unpack arrays and any other iterables:

let [firstName, lastName] = person

Note: Destructuring is not destructive. Destructuring does not change the original object.
